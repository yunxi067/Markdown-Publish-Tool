# Contributing

Thanks for your interest in contributing.

## Development

```bash
npm install
npm run dev
```

## Before Opening a Pull Request

- Keep changes focused and small when possible
- Follow the existing project structure and naming style
- Run `npm run build`
- If you add tests, run `npm test`

## Pull Requests

- Describe what changed and why
- Include screenshots for UI changes when helpful
- Note any known limitations or follow-up work

## Discussions

For larger changes, open an issue or discussion first so the implementation direction can be aligned early.
